# ESG 碳资产认证工作流示例

## 示例 1: 林业碳汇项目认证

### 输入
```
我有一个1000公顷的林业碳汇项目，需要申请CCER认证，请帮我完成整个流程
```

### 执行步骤

1. **需求分析**
   - 项目类型：林业碳汇
   - 土地面积：1000公顷
   - 树种：杉木、马尾松
   - 预计年减排量：15000吨CO2

2. **方法学选择**
   - 适用方法学：CCER-01 造林碳汇
   - 计入期：30年

3. **文档准备**
   - 项目设计文件（PDD）
   - 土地权属证明
   - 造林方案
   - 监测计划

4. **申请提交**
   - 提交至：生态环境部
   - 第三方核查机构：XXX

### 输出
- 项目注册号
- 年减排量证书
- 交易建议

## 示例 2: 光伏发电项目认证

### 输入
```
我有一个50MW的光伏发电项目，需要申请CCER认证
```

### 执行步骤

1. **项目信息收集**
   - 装机容量：50MW
   - 位置：内蒙古
   - 并网时间：2024年1月
   - 预计年减排量：80000吨CO2

2. **方法学匹配**
   - 适用方法学：CCER-06 并网光伏发电

3. **减排量核算**
   - 基准线排放：0.5819 tCO2/MWh
   - 项目排放：0 tCO2/MWh
   - 减排量：0.5819 tCO2/MWh
   - 年发电量：75000 MWh
   - 年减排量：43642.5 tCO2

## 示例 3: 碳排放报告生成

### 输入
```
帮我分析公司2025年的碳排放情况，并生成报告
```

### 执行步骤

1. **数据收集**
   - 范围1排放：直接排放
   - 范围2排放：外购电力
   - 范围3排放：供应链

2. **排放核算**
   - 燃烧排放计算
   - 电力排放因子

3. **报告生成**
   - 年度排放总量
   - 排放强度
   - 减排建议
